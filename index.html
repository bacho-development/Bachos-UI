-- Пример Lua кода
function factorial(n)
    if n == 0 then
        return 1
    else
        return n * factorial(n - 1)
    end
end

print("Факториал 5:", factorial(5))

О нашем проекте

Это демонстрационное приложение создано для показа возможностей табов в Bootstrap

Основные возможности:

  • Переключение между вкладками
  • Сворачиваемая боковая панель
  • Вложенные табы в дропдаунах
  • Навигация внутри контента
  • Подсветка кода
Технологии

Bootstrap 5, JavaScript, HTML5, CSS3
